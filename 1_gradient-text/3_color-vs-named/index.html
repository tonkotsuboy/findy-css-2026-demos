color(srgb …) を使うだけで自動 Oklab 補間

どちらも in <色空間> を書いていない。
色の書き方が違うだけで、補間色空間が自動で切り替わる。

blue → red

named
/ hex

補間 = sRGB（中央: 暗い灰紫）

color()
srgb

補間 = Oklab（中央: 鮮やかな紫）

green → magenta

named
/ hex

補間 = sRGB（中央: 濁ったグレー）

color()
srgb

補間 = Oklab（中央: 鮮やかな黄緑〜ピンク）

yellow → blue

named
/ hex

補間 = sRGB（中央: くすんだグレー）

color()
srgb

補間 = Oklab（中央: 鮮やかな緑寄り）

red → cyan

named
/ hex

補間 = sRGB（中央: 灰色）

srgb
